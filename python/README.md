# PaddleOCR-TVM Python

Standalone Python package scaffold for this repository, managed with `pyproject.toml` and Pixi.

## Quickstart

```bash
pixi run start
pixi run -e dev check
```

## Available tasks

- `pixi run start`
- `pixi run -e dev format`
- `pixi run -e dev lint`
- `pixi run -e dev typecheck`
- `pixi run -e dev test`
- `pixi run -e dev test-cov`
- `pixi run -e dev check`
